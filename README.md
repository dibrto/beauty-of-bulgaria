# Красотата на България

## Цел
Красотата на България

## Използвани технологии
- HTML5 със семантични елементи (header, nav, main, section, article, footer)
- CSS3 (адаптивен дизайн, grid/flexbox)
- JavaScript (мобилно меню, падащо меню, галерия/лайтбокс, бутон „нагоре")

## Структура на проекта
- `index.html` — начална страница с обща информация и мини галерия
- `mountains.html`, `waterfalls.html`, `caves.html` — вътрешни страници с описания и галерии
- `css/styles.css` — всички стилове
- `js/main.js` — скриптове за интерактивност
- `docs/project_description.md` — настоящата документация.
